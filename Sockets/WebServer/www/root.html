You can make the following GET requests

  • /file/sample.html -- returns the content of the file sample.html
  • /json -- returns a json of the /random request
  • /random -- returns index.html
  • /multiply?num1=[INTEGER]&num2=[INTEGER] -- multiplies 2 integers
  • /madlibs?word1=[STRING]&word2=[STRING] -- puts 2 words in a random sentence
  • /rps?player1=[rock|paper|scissors]&player2=[rock|paper|scissors] -- Decides outcome of Rock, Paper, Scissors game between players
